# week1-git-practical1

CRUD operations on weather data of Gandhinagar for past 10 days in Python.

## Description
This project demonstrates basic CRUD (Create, Read, Update, Delete) operations using a Python program that analyzes weather data of Gandhinagar.

## Features
- Calculate **average** and **median** for:
  - Temperature
  - Humidity
  - AQI
- Store results in a text file
- Perform file deletion as part of CRUD operations

## Files
- `weather_analysis.py` – Main Python script
- `results.txt` – Output file (generated and later deleted)

## How to Run
```bash
python weather_analysis.py
